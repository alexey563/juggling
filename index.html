Инструкция по использованию

🖱️ Управление камерой

  • ЛКМ + перетаскивание: Поворот камеры вокруг сцены
  • ПКМ + перетаскивание: Перемещение камеры
  • Колесо мыши: Приближение/отдаление
  • На мобильном: Жесты касания для управления камерой

👤 Работа с объектами

  • Добавление: Используйте кнопки "Добавить жонглёра" и "Добавить куб"
  • Выделение: Кликните по объекту для выделения (желтая рамка)
  • Перемещение: ЛКМ на объект → объект становится полупрозрачным → повторный ЛКМ для установки
  • Поворот: При перемещении крутите колесо мыши для поворота на 45°
  • Размещение: Объекты можно ставить друг на друга

🔄 Перекидки

  • Создание: Включите "Режим создания перекидок" → кликните на двух жонглёрах
  • Высота: ЛКМ на линию перекидки → движение мыши вверх/вниз → повторный ЛКМ
  • Количество: Настройте количество реквизита перед созданием
  • Пересечения: Желтые шарики показывают места пересечения перекидок

💾 Сценарии

  • Сохранение: Введите название → "Сохранить сценарий"
  • Загрузка: Кликните по миниатюре в левой панели
  • Удаление: Нажмите "×" на миниатюре сценария
  • Автоскриншот: При сохранении создается миниатюра сцены

📱 Мобильное управление

  • Панели: Используйте кнопки 📁 и 🔧 для открытия панелей
  • Быстрые действия: Кнопки внизу экрана для основных операций
  • Касания: Одиночное касание для выделения, двойное для перемещения
  • Жесты: Пинч для масштабирования, свайп для поворота камеры
