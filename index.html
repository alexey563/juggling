🚀 Быстрый старт

  • Тестовая сцена: Нажмите "Создать тестовую сцену" для знакомства с программой
  • Первая анимация: Добавьте объекты → "📍 Добавить ключевой кадр" → переместите объекты → "📍 Добавить ключевой кадр" → "▶ Запустить анимацию"

🖱️ Управление камерой

  • ЛКМ + перетаскивание: Поворот камеры вокруг сцены
  • ПКМ + перетаскивание: Перемещение камеры
  • Колесо мыши: Приближение/отдаление
  • На мобильном: Жесты касания для управления камерой

👤 Работа с объектами

  • Добавление: Кнопки "+ Жонглёр" и "+ Куб"
  • Настройка размеров куба: Измените ширину, высоту, длину в правой панели перед добавлением
  • Выделение: Клик по объекту (появится желтая рамка)
  • Перемещение: Зажмите объект на 800ms без движения мыши → объект станет полупрозрачным → перемещайте → отпустите
  • Поворот камеры: Зажмите объект + сразу двигайте мышь (для поворота камеры)
  • Поворот объекта: ПК - колесо мыши при перемещении; Мобильный - кнопки "↺ Повернуть"
  • Размещение: Объекты можно ставить друг на друга, система автоматически найдет правильную высоту
  • Удаление: Выделите объект → кнопка "Удалить выбранное"

🔄 Создание перекидок

  • Активация режима: Нажмите кнопку "Перекидки" (кнопка станет красной)
  • Настройка количества: Установите количество реквизита (1-10) в поле "Количество реквизита"
  • Создание перекидки: Кликните на первом жонглёре, затем на втором
  • Изменение высоты: Зажмите линию перекидки на 800ms → двигайте мышь вверх/вниз → отпустите
  • Выделение перекидки: Клик по линии (появится желтое свечение)
  • Удаление перекидки: Выделите перекидку → "Удалить выбранное"
  • Пересечения: Желтые шарики автоматически показывают места пересечения перекидок
  • Выход из режима: Нажмите кнопку "Перекидки" еще раз

💾 Сохранение и загрузка сценариев

  • Сохранение: Введите название в поле "Название сценария" → "Сохранить сценарий"
  • Что сохраняется: Все объекты, перекидки, полная анимация со всеми кадрами, скорость анимации
  • Загрузка: Кликните по миниатюре сценария в левой панели
  • Удаление сценария: Нажмите "×" на миниатюре сценария
  • Экспорт: "Экспорт сценариев" - сохраняет все ваши сценарии в файл
  • Импорт: "Импорт сценариев" - загружает сценарии из файла
  • Автоскриншот: При сохранении автоматически создается миниатюра сцены
  • Персональные сценарии: Каждый пользователь видит только свои сценарии

🎬 Создание анимации

  • Основной принцип: Кадр сохраняет снимок всех объектов в текущий момент
  • Создание кадра: Расставьте объекты → "📍 Добавить ключевой кадр"
  • Появление объектов: Добавьте новые объекты → создайте кадр (объекты появятся в этом и последующих кадрах)
  • Исчезновение объектов: Удалите объекты → создайте кадр (объекты исчезнут из этого и последующих кадров)
  • Редактирование кадра: Кликните по названию кадра в списке → измените объекты → "🔄 Обновить кадр"
  • Запуск анимации: Минимум 2 кадра → "▶ Запустить анимацию"
  • Скорость: Настройте слайдером (0.5x - 3.0x)
  • Остановка: "⏸ Остановить анимацию"
  • Циклическое воспроизведение: Анимация автоматически повторяется
  • Удаление кадров: Кнопка "×" рядом с кадром или "🗑️ Очистить анимацию"

📋 Пример создания сложной анимации

  • Кадр 1: Добавьте 3 жонглёра и 2 куба → создайте кадр
  • Кадр 2: Переместите жонглёров, удалите 1 куб → создайте кадр
  • Кадр 3: Удалите 1 жонглёра, добавьте новый куб в другом месте → создайте кадр
  • Кадр 4: Добавьте нового жонглёра, создайте перекидки → создайте кадр
  • Результат: Объекты плавно перемещаются, появляются и исчезают по ходу анимации

📱 Мобильное управление

  • Панели: Кнопки 📁 (сценарии) и 🔧 (инструменты) для открытия панелей
  • Нижняя панель: Быстрый доступ ко всем основным функциям
  • Анимация: "▶ Анимация", "📍 Кадр", "� О"бновить" в нижней панели
  • Касания: Короткое касание для выделения, долгое касание (800ms) для перемещения
  • Жесты: Пинч для масштабирования, свайп для поворота камеры
  • Поворот объектов: Кнопки "↺ Повернуть" и "↻ Повернуть" в нижней панели
